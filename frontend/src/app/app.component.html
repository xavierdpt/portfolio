Angular

Angular est l'un des framework de développement web les plus à la mode en 2018.

Que faut-il savoir sur Angular quand on est maître d'oeuvre d'un projet informatique ?

Tout d'abord, il faut savoir qu'Angular est le successeur d'AngularJS.

La première version d'AngularJS a été mise à disposition en février 2010.

La première versoin d'Angular a été mise à disposition en septembre 2016.

Les deux versions sont très différentes l'une de l'autre. Bien qu'AngularJS soit toujours maintenue, il est fortement conseillé d'utiliser Angular dans de nouveaux projets.

Il est aussi intéressant de savoir qu'Angular et AngularJS ont été conçus par Google.

Le but d'Angular est de simplifier le développement de projets web.

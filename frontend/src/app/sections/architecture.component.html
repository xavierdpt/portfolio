What is a software architect ?

A software architect, in general, is a software expert who has the ability to make decisions involving technical standards, tools and platforms.

What is a Java EE architect ?

A Java EE architect is specialized to the Java EE world, which is a small universe. He knows in details the Java EE specifications, the characteristics and abilities of different application servers, and how to use and integrate various Java libraries and tools to achieve some purpose.
